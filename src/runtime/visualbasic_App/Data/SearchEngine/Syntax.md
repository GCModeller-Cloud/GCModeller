+ Normal search, **contains any token**: ``term``
+ The result **MUST** contains this term: ``"this term"``
+ **AND**: ``term1 AND term2``
+ **OR**: ``term1 OR term2``
+ **NOT**: ``NOT term3``
+ Search **in Field**: ``<field>: expression``

Please note that, the operator is case sensitive, all of them should be **UPCASE**
